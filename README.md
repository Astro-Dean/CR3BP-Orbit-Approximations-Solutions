# Analytical Solutions for Quasi-Periodic Lissajous/Halo Orbits at $L_{2}$ Lagrangian Libration Point
This repository contains algorithms and solutions for computing quasi-periodic lissajous and halo orbits in the Circular Restricted Three-Body Problem (CR3BP) environment around $L_2$ Lagrangian libration point.
